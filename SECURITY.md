# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| `main`  | Yes       |

## Scope

This project is a static frontend website with no backend, API, authentication, or user data processing. The attack surface is limited to the client-side code served via GitHub Pages.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly:

1. **Do not** open a public GitHub issue.
2. Send an email to **[info@fat-music-official.de]** with a description of the vulnerability.
3. You will receive a response within 7 days.

## Out of Scope

- Issues in third-party dependencies (report these to the respective maintainers)
- Denial of service attacks against GitHub Pages
- Social engineering
